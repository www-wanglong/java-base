## 基本数据类型
CPU可以直接进行运算的类型。

计算机内存的最小存储单元是字节（byte），一个字节就是一个8位二进制，即8个bit。它的二进制范围是00000000 ～ 11111111，十进制就是0 ～ 255
### 整数类型：byte、short、int、long
- byte：-128 ~ 127
- short: -32768 ~ 32767
- int: -2147483648 ~ 2147483647
- long: -9223372036854775808 ~ 9223372036854775807
### 浮点数据类型：float、double
- float 4个字节
- double 8个字节
### 字符类型：char
- 2个字节
### 布尔类型： boolean
